#  Flavora — QA Testing Documentation

##  Layihə Haqqında

Bu repository *Flavora* platformasının test sənədlərini əhatə edir.  
Flavora istifadəçilərin resept paylaşa, cooking session yarada və digər istifadəçilərlə birlikdə yemək bişirə bildiyi platformadır.

Platforma həmçinin sağlam qidalanma və sağlam həyat tərzi haqqında məlumat təqdim edir.

---

##  Platforma Haqqında

Flavora istifadəçilərə aşağıdakı imkanları təqdim edir:

- Resept paylaşmaq
- Cooking session yaratmaq
- Digər istifadəçilərlə birlikdə yemək bişirmək
- Sağlam qidalanma haqqında məlumat almaq
- Sağlam həyat tərzi ilə bağlı tövsiyələr əldə etmək
- İstifadəçi profili yaratmaq və idarə etmək

---

##  Test Sənədləri

Bu layihə çərçivəsində aşağıdakı test sənədləri hazırlanmışdır:

- User Story

- Test Checklist  
- Test Cases  
- Test Summary Report
